# SQL30 - SQL in Marketing Cloud

Sample datasets for the [SQL30 30-day challenge](https://marketingcloud30.com/challenges/), where you practice writing SQL against realistic Marketing Cloud data for the fictional telecom company **NextMobile**.

## What's here

A set of CSV files you import as data extensions, then query as the daily challenges direct. Each file is one dataset (users, packages, usage, loyalty, services, and more) and is numbered to match the day it's introduced.

This repo holds the **data only** - the scenarios and solutions live on the website.

## How to use

1. Sign up for SQL30 at [marketingcloud30.com](https://marketingcloud30.com/challenges/).
2. Import the CSV(s) a day references as data extensions in your org.
3. Write your query and check it against the solution on the challenge page.
