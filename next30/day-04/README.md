# Day 4 - Consent Management

Create subscription channels, publish a preference page, and import a first batch of leads with their consent status set.

## Contents
| File | Purpose |
|------|---------|
| `leads.csv` | First batch of leads to import (First Name, Last Name, Email, Company, Country). |
| `consent-optin.csv` | Consent opt-in dates for those leads (Email, Consent Date). |

## How to use
1. Import `leads.csv`.
2. Set each lead's consent using `consent-optin.csv`.
