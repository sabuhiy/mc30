## How it works

1. **Sign up** at [marketingcloud30.com](https://marketingcloud30.com/challenge/next30/) and pick Next30.
2. **Get one email a day** for 30 days, each with a short business scenario and a task to complete in your own Marketing Cloud org.
3. **Build it yourself.** Most days you build directly in the platform. Some days need a starter file (a CSV, an image, sample copy) - those live in this repository, one folder per day.
4. **Check your work** against the solution and the interactive walkthrough on the challenge page.

> This repo holds **only the starter resources** you download to complete a day. The scenarios, step-by-step solutions, and walkthroughs live on the website - nothing here spoils the answer.

## What each day folder contains

Every `day-NN/` folder has a short `README.md` describing that day's focus and listing any files to download. If a day builds entirely inside your org, its folder says so and has no extra files.
