## What this is

[MarketingCloud30](https://marketingcloud30.com/) is a community learning platform. You sign up for a 30-day program, pick a challenge, and each day you receive an email with a short scenario and a task to complete in your own Marketing Cloud org. Every challenge is framed as a real marketing problem, so you practice the skill the way you'd actually use it on the job.

**This repository holds the starter resources** for those challenges - the sample data, images, and copy you download to complete a day. The scenarios, solutions, and interactive walkthroughs live on the website, so nothing here gives away the answer.

## How to use this repo

1. Sign up for a program at [marketingcloud30.com](https://marketingcloud30.com/challenges/) and start receiving the daily emails.
2. When a day needs a starter file, open that program's folder here (for example [`next30/day-03`](next30/day-03)) and download what the day's `README.md` lists.
3. Build the challenge in your org, then check your work against the solution on the challenge page.
